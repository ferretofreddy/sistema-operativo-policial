# Documentación Oficial SOP

Estructura documental oficial del proyecto:

- `index/`: índice maestro y orden de lectura.
- `architecture/`: arquitectura y diseño técnico.
- `database/`: reglas y criterios de base de datos.
- `operational/`: reglas de dominio y estado operativo.
- `issues/`: incidencias conocidas y fixes históricos.
- `skills/`: guías de ejecución para asistentes IA.
